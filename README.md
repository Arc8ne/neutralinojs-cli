# neutralinojs-cli

neu cli for Neutralinojs

```
  $ npm i -g @neutralinojs/neu
```

Synopsis

- `neu create myapp --template <templateName>`
- `neu build`
- `neu run`
- `neu release`
- `neu update`

Supported templates

Generic

- Javascript `--template js`
- Typescript `--template ts`
- Kotlin `--template kotlin`

Frameworks/Libraries

- React `--template react`
